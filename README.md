# 图像分类训练工具 —— 使用说明

这是一款**零代码基础、可视化操作**的图像分类 AI 工具，支持**自动训练模型**和**一键图片预测**，无需编写代码，小白也能快速完成图像识别任务。

## 一、核心功能

1. **模型训练**：支持两种数据集分类方式，自动训练高精度图像分类模型，实时显示训练日志和准确率曲线
2. **图片预测**：加载训练好的模型，对任意图片进行智能识别，显示分类结果和置信度
3. 自动保存最优模型，支持训练中途停止，可视化图表直观展示训练效果

------

## 二、环境准备（首次使用必看）

### 1. 安装依赖库

打开电脑的**命令提示符（CMD）/ 终端**，复制粘贴以下命令并执行：

bash

```
pip install torch torchvision pillow pyqt5 matplotlib
```

### 2. 运行软件

直接运行你提供的代码文件，软件窗口会自动打开。

------

## 三、详细使用步骤

### 模块一：🧠 模型训练（第一步：训练专属 AI 模型）

#### 1. 参数设置（新手直接用默认值即可）

表格

| 参数       | 说明                         | 推荐值                             |
| :--------- | :--------------------------- | :--------------------------------- |
| 批次大小   | 每次训练加载的图片数量       | 8~16（电脑配置低就调小）           |
| 训练轮数   | 模型学习的次数               | 5~20（轮数越多精度越高，耗时越长） |
| 学习率     | 模型学习速度                 | 0.001（无需修改）                  |
| 分类模式   | **核心设置**：两种数据集格式 | 二选一                             |
| 测试集比例 | 用于测试精度的图片比例       | 0.2（无需修改）                    |
| 数据路径   | 数据集文件夹路径             | 选择你的图片文件夹                 |

#### 2. 数据集准备（两种格式任选一种）

##### 格式 1：按文件夹分类（推荐，支持多级文件夹）

- 新建一个总文件夹（如：`花卉数据集`）
- 总文件夹内创建**子文件夹**，文件夹名称 = 分类名称
- 把对应类别的图片放入子文件夹

plaintext

```
花卉数据集/
├─ 玫瑰/
│  ├─ 1.jpg
│  └─ 2.png
├─ 百合/
│  └─ 3.jpg
```

##### 格式 2：按图片名称分类

- 所有图片放在**同一个文件夹**

- 图片名称包含分类标签（汉字 / 英文均可）

  

  例：

  ```
  猫_01.jpg
  ```

  ```
  狗02.png
  ```

  ```
  汽车123.jpeg
  ```

#### 3. 开始训练

1. 点击【选择数据集】，选中你的数据集文件夹
2. 点击【开始训练】，软件自动开始训练
3. 实时查看：**训练日志**（下方文本框）+ **训练曲线**（损失 / 训练精度 / 测试精度）
4. 最优模型会**自动保存**到软件目录下的 `best_model` 文件夹中

点击【停止训练】可随时终止训练，已保存的最优模型不会丢失。

### 模块二：🔍 图片预测（第二步：用模型识别图片）

#### 1. 加载训练好的模型

1. 切换到「图片预测」标签页
2. 点击【加载模型】
3. 选择 `best_model/best_model.pth` 文件
4. 弹出提示框显示分类类别，代表加载成功

1. 点击【选择图片预测】
2. 选择你要识别的图片（支持 jpg/png/bmp）

## 四、常见问题与注意事项

### 1. 训练报错 / 无法启动

- 检查**数据路径**是否为纯英文 / 中文（不要有特殊符号、空格）
- 检查图片格式：仅支持 `jpg/jpeg/png/bmp`
- 批次大小调小（电脑配置低建议设为 8）

### 2. 精度太低怎么办

- 增加**训练轮数**（比如从 5 轮改成 20 轮）
- 增加数据集图片数量（每类至少 20 张以上）
- 保证图片清晰，分类标签准确

### 3. 模型文件说明

- 训练完成后，`best_model/best_model.pth` 是唯一需要保存的模型文件
- 迁移软件时，只需复制这个文件即可在其他电脑上预测

### 4. 系统兼容

- 支持 Windows/macOS
- 无需独立显卡，CPU 即可正常运行

## 五、快速上手示例（新手必看）

1. 准备 10 张猫图片、10 张狗图片，分两个文件夹存放
2. 打开软件，选择这个文件夹，默认参数直接点击【开始训练】
3. 训练完成后，加载模型，随便选一张猫 / 狗图片预测
4. 一秒得到识别结果，完成你的第一个 AI 图像分类项目！
